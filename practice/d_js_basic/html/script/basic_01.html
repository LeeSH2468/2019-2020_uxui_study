javascript란?

자바스크립드란, 객체기반의 스크립트 프로그래밍 언어.
주로 웹에 특화되어있는 프로그래밍 언어이다. 과거, 자바의 아류이자 초급언어라는 저급한 언어라는 칭호를 들은 적이 있지만, 현재(2019년)는 가장 인기있는 언어중의 하나이다.

객체기반, 객체/함수지향, 일급객체로 불리우는 언어이며,
과거 웹에서 HTML,CSS만 제어하던 시절을 넘어서 현재 앱을 제작하기도 하며, 서버를 컨트롤하기도 하는 다양한 분야에서 쓰이는 언어.

자바스크립트는 ECMAscript를 표준언어 체계로 따르고 있으며, ECMA표준은 매년 6월 17일에 새로운 표준을 선언하고 있음.

유사 언어는,
Type Script, Dart, Elm, Coffee Script(현재사용x.js와 다르게 바뀌어서 사용x), Action Script(현재사용x.플래시시절 많이 사용) 등이 있다.

자바스크립트에서 인기있는 라이브러리, 프레임워크 jQuery, AngularJs, ReactJs, VueJs, handlebarJs, ember, polymer, d3Js ...
